#### 💠 Installation 💠 
EF-Tools is part of pyRevit default Extensions. 
And it has very simple and quick installation.  

    ▶ Install pyRevit  
    ▶ Open Extensions Menu  
    ▶ Select EF-Tools  
    ▶ click on Install

🎬 YouTube Tutorial - https://www.youtube.com/watch?v=q0zbdyyBIo0
